# Hawaa Branding Manifest

## Brand direction

- Visual tone: professional, calm, financial, Arabic-friendly.
- Primary concept: ledger/accounting card + financial trend mark.
- Primary color: deep teal `#0f766e`.
- Dark primary: `#134e4a`.
- Accent: amber `#d97706`.

## Assets

| File | Usage |
|---|---|
| `app.ico` | Windows EXE, taskbar, main window |
| `installer.ico` | Windows installer |
| `project_file.ico` | Optional `.hawa` file association |
| `backup_file.ico` | Optional backup-file icon |
| `app_icon_*.png` | Runtime fallback icons and UI display |
| `app_logo.png` | Splash/sidebar/logo display |
| `app_logo.svg` | Scalable source logo |
| `app_symbol.svg` | Scalable source symbol |

## Rules

- Keep the icon simple at 16×16 and 32×32. Do not add text inside the app icon.
- Use the full logo only in splash/sidebar/report covers.
- Use the symbol for EXE/taskbar/shortcut/file icons.
- Keep visual effects light: fade, slide, hover, and small shadow only.
