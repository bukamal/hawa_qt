# Hawaa Sound Manifest

Short WAV feedback sounds for business events.

- success.wav — save/complete
- error.wav — validation/system error
- warning.wav — unsaved changes/payment warning
- delete.wav — destructive action
- notify.wav — informational notification
- backup_done.wav — backup completed
- export_done.wav — report export completed
- login_ok.wav — successful login
- login_fail.wav — failed login
- server_on.wav — local server started
- server_off.wav — local server stopped
- payment_due.wav — payment reminder/overdue notice

All sounds are intentionally short, office-safe, and optional.
