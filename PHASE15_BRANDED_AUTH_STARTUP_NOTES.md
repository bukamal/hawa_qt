# Phase 15 — Branded Auth & Startup Experience

## الهدف
توحيد تجربة ما قبل الواجهة الرئيسية تحت هوية المشروع: شاشة التشغيل، التفعيل، تسجيل الدخول، والانتقال بعد تسجيل الدخول.

## ما تم

### 1. شاشة تسجيل الدخول
تم تحديث `views/login_dialog.py` لتستخدم شعار المشروع واسم النظام ونفس لوحة الألوان. أصبحت الشاشة تعرض:

- شعار هوى الشام.
- وصف مختصر لتسجيل الدخول.
- شارة وضع التشغيل: محلي أو شبكي.
- بطاقة بيانات دخول واضحة.
- رسالة خطأ Inline بدل رسالة جامدة.
- صوت نجاح/فشل تسجيل الدخول.
- تلميح آمن لأداة إعادة تعيين كلمة المرور.

### 2. واجهة التفعيل
تم تحديث `views/activation_dialog.py` لتصبح شاشة تفعيل تشخيصية لا مجرد Dialog بسيط. أصبحت تعرض:

- شعار المشروع.
- حالة ترخيص البرنامج.
- حالة ترخيص الشبكة.
- معرّف الجهاز مع زر نسخ.
- مكان ملف الترخيص.
- تشخيص سبب رفض الترخيص.
- زر فتح مجلد الترخيص.
- زر تحديث الحالة.
- رسائل نجاح/خطأ واضحة.

### 3. شاشة الانتظار Startup
تم تحديث `views/splash_screen.py` لتعرض مراحل تشغيل واضحة:

- تهيئة المسارات.
- تحميل الهوية البصرية.
- فحص قاعدة البيانات.
- فحص الترخيص.
- تسجيل الدخول.
- تجهيز الواجهة.

كما أصبحت تدعم `set_error()` لإظهار فشل التشغيل بدل السقوط الصامت.

### 4. واجهة ما بعد تسجيل الدخول
تم تعديل `main.py` بحيث تظهر شاشة الانتظار بعد نجاح تسجيل الدخول وقبل فتح الواجهة الرئيسية. تعرض:

- ترحيبًا بالمستخدم.
- تحميل الصلاحيات.
- تجهيز لوحة التحكم والحسابات والتقارير.

### 5. مكونات مشتركة
أضيفت:

```text
ui/auth/brand_panel.py
ui/auth/startup_helpers.py
ui/auth/__init__.py
```

هذه الملفات تفصل عناصر الهوية الخاصة بما قبل تسجيل الدخول عن باقي الواجهة.

### 6. تشخيص الترخيص
أضيفت في `auth/activation.py`:

```text
get_license_file_paths()
describe_license_state()
```

حتى لا تضطر الواجهة إلى قراءة ملف الترخيص أو فكّه بنفسها.

## ملاحظات اختبار Windows
يجب اختبار هذه النقاط يدويًا:

1. هل تظهر شاشة التشغيل بالشعار قبل تسجيل الدخول؟
2. هل تعرض شاشة التفعيل حالة الترخيص ومسار الملف ومعرّف الجهاز؟
3. هل تسجيل الدخول يعرض الشعار بدل النص القديم؟
4. هل تظهر شاشة انتقال قصيرة بعد تسجيل الدخول وقبل الواجهة الرئيسية؟
5. هل أصوات الدخول الناجح والخاطئ تعمل؟
6. هل لا يعود البرنامج لطلب التفعيل بعد إعادة التشغيل؟
